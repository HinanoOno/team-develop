就活エージェント比較サイトとは？

ーAbout this siteー

就活エージェントとは？：就活のプロと対話しながらそれぞれの学生に合った企業を探すサービスのこと

比較サイトとは？：エージェントを絞り込み検索で比較するサイト


このサイトでは、数あるエージェントの中からあなたに合ったエージェント選びをサポートします。

How to Use CRAFT

検索

自分の専攻・興味のあるエージェ
ントで絞り込み、検索してみる。

エージェントをクリック

気になったのを何個か選んで、カートに入れる。

(1度個人情報を登録すれば、エージェントの一括申請が可能です)

応募

個人情報登録で応募完了。
後日、エージェントから連絡がきます。
